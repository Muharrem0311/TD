1. La déclaration DOCTYPE est elle obligatoire ?
    • Réponse: non mais il est préférable pour que le navigateur sache quelle langage est utlisé
  2. Où doit-elle être placée ?
    • Réponse: Elle doit être placé au tout début
  3. Quelles informations contient-elle ?
    • Réponse: le langage utilisé pour le code
  4. Que faut-il faire pour que le code HTML5 soit validé ?
    • Réponse: il faut enregistrer le fichier en .html
  5. A quel emplacement est défini le jeu de caractère utilisé dans le document ?
    • Réponse: il est utilisé dans l'emplacement "head"
  6. Où est déclaré le nom de l'élément racine du document ?
    • Réponse: après le DOCTYPE
  7. Quel est son rôle et son nom ?
    • Réponse: il permet de faire lire le code au navigateur
  8. Quels sont les élément enfants de l'élément "html" ?
    • Réponse: les éléments enfants de l'élément "html" sont "head" et "body"
